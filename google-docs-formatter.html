Installation Guide

Add the formatter to your Google Docs in under 2 minutes:

1

Open Google Docs

Create a new document or open an existing one at docs.google.com

2

Open Apps Script

Click Extensions in the menu bar, then select Apps Script

3

Delete Default Code

Select all the default code in the editor (Ctrl+A / Cmd+A) and delete it

4

Copy & Paste the Script

Copy the entire code below and paste it into the Apps Script editor

5

Save the Project

Press Ctrl+S (or Cmd+S on Mac) and name it "FireICE Formatter"

6

Refresh Your Document

Go back to your Google Doc and refresh the page. You'll see a new FireICE Format menu!

Note: The first time you use the formatter, Google will ask you to authorize the script. This is normal - click "Continue" and grant the necessary permissions.

How to Use

Formatting Text

Select the text you want to format, then choose an option from the FireICE Format menu. The formatting will be applied to your selection.

Inserting Templates & Layouts

Place your cursor where you want to insert content, then select a template or layout from the menu. The content will be inserted at your cursor position.

Applying Color Schemes

Color schemes affect the entire document based on heading styles. Use Google Docs' built-in heading styles (Title, Heading 1, etc.) for best results, then apply a color scheme.

Tip: The templates are fully editable after insertion - just replace the placeholder text with your own content!

Troubleshooting

Menu not appearing?

Make sure you saved the script and refreshed your Google Doc. The menu only appears after a page refresh.

Authorization error?

Click "Advanced" then "Go to FireICE Formatter (unsafe)" to proceed. Google shows this warning for all custom scripts.

"Please select text" message?

Make sure you've selected (highlighted) text before applying text formatting options.
